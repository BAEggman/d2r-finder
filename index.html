    시세 연동 설정 — Cloudflare Worker 프록시
    
시세를 이 페이지 안에 표시하려면 무료 Cloudflare Worker 프록시가 하나 필요합니다 (브라우저가 트레더리를 직접 부르지 못하게 막혀 있어서요). 만드는 방법은 README에 5분 가이드로 적어 두었습니다. 배포한 Worker 주소를 여기 저장해 주세요.

    

      
      
    

    

  


  

    
시세 표시 — ‘시세 불러오기’는 프록시를 거쳐 트레더리 매물을 이 페이지에 보여줍니다. 호가는 참고용이며, 실제 거래·판매자 연락은 ‘Traderie에서 보기’로 이동해 진행하세요.

    
래더 · 플랫폼 — 페이지 안 시세에는 여러 모드의 매물이 섞여 있을 수 있습니다. 매물별 표시 정보를 확인하고, 정밀 필터(래더/스탠, 소프트/하드, PC/콘솔)는 트레더리 화면 상단에서 선택하세요.

    
참고 — 시세 연동은 트레더리의 비공식 API를 사용하므로 사이트 개편 시 예고 없이 멈출 수 있습니다. 그 경우에도 검색·링크 기능은 계속 동작합니다. 이 도구는 팬 유틸리티이며 Blizzard·Traderie와 무관합니다.
